# 🌍 Language Translation Tool

A simple web app to translate text between multiple languages.  
Built during my **CodeAlpa Internship — Project 2**

---

## Tech Used
- Python + Flask
- deep-translator (Google Translate)
- HTML, CSS, JavaScript

---

## How to Run

```bash
pip install -r requirements.txt
python app.py
```

Then open `http://127.0.0.1:5000` in your browser.

---

## Features
- Supports 10+ languages
- Auto language detection
- Copy & swap language buttons
- Error handling
